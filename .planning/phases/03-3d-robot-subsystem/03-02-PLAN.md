**File 1: `src/components/robot/RobotLoadingIndicator.tsx`** (ROBT-03)
- `'use client'` directive
- Import `useProgress` from `@react-three/drei` -- BUT this file will be imported from RobotCanvas.tsx which is the SSR boundary. `useProgress` is a Zustand store internally and CAN be imported outside Canvas. However, to keep the SSR boundary clean, create TWO versions:
  - A **static fallback** version (used as `loading` prop of dynamic()) that does NOT import drei. Shows a simple "Loading 3D Model..." text with a pulsing animation using Tailwind classes.
  - The **progress overlay** version is rendered INSIDE RobotCanvas.tsx (in DOM space, outside Canvas) and DOES use `useProgress` from drei.
- Actually, the simpler approach per the research: `useProgress` works outside Canvas (it's a module-level Zustand store). BUT the dynamic import boundary file (RobotCanvas.tsx) MUST NOT import drei. So put the progress overlay in a separate file that is dynamically imported alongside RobotScene.

Best approach:

**File 2: `src/components/robot/RobotModel.tsx`** (ROBT-01, ROBT-02)
- `'use client'` directive
- Imports: `useRef`, `useEffect` from react; `useGLTF`, `useAnimations` from `@react-three/drei`; `Group` from `three`; `useRobotStore` from `@/stores/useRobotStore`; `EMOTION_CLIP_MAP`, `MODEL_PATH`, `RobotEmotion` from `@/types/robot`
- Component `RobotModel`:
  1. `const groupRef = useRef<Group>(null)`
  2. `const { scene, animations } = useGLTF(MODEL_PATH)`
  3. `const { actions } = useAnimations(animations, groupRef)`
  4. Subscribe to emotion from store: `const emotion = useRobotStore((s) => s.emotion)`
  5. Subscribe to setModelLoaded: `const setModelLoaded = useRobotStore((s) => s.setModelLoaded)`
  6. On mount, call `setModelLoaded(true)`. On unmount, call `setModelLoaded(false)`.
  7. Log animation clip names on mount for debugging: `console.log('[Robot] Available animations:', animations.map(a => a.name))` -- this is critical for when the real model is swapped in.
  8. Emotion-driven animation crossfade:
     ```tsx
     const prevEmotionRef = useRef<RobotEmotion>(emotion);
     useEffect(() => {
       const prevClip = EMOTION_CLIP_MAP[prevEmotionRef.current];
       const nextClip = EMOTION_CLIP_MAP[emotion];
       const prevAction = actions[prevClip];
       const nextAction = actions[nextClip];
       if (nextAction) {
         if (prevAction && prevAction !== nextAction) {
           prevAction.fadeOut(0.5);
         }
         nextAction.reset().fadeIn(0.5).play();
       }
       prevEmotionRef.current = emotion;
     }, [emotion, actions]);
     ```
  9. Return `<group ref={groupRef}><primitive object={scene} /></group>`
  10. After the component: `useGLTF.preload(MODEL_PATH)` to start download early.

  ```tsx
  function LoadingOverlay() {
    const { progress, active } = useProgress();
    if (!active) return null;
    return (
      <div className="absolute inset-0 z-10 flex items-center justify-center bg-surface-base">
        <div className="text-center">
          <div className="font-display text-lg text-accent">
            Loading 3D Model...
          </div>
          <div className="mt-3 h-1 w-48 overflow-hidden rounded bg-surface-overlay">
            <div
              className="h-full bg-accent transition-all duration-300"
              style={{ width: `${progress}%` }}
            />
          </div>
          <div className="mt-2 text-sm text-text-muted">
            {Math.round(progress)}%
          </div>
        </div>
      </div>
    );
  }
  ```
- Main component `RobotScene` (default export):
  ```tsx
  export default function RobotScene() {
    const setPerformanceTier = useRobotStore((s) => s.setPerformanceTier);
    return (
      <>
        <LoadingOverlay />
        <Canvas
          dpr={[1, 2]}
          camera={{ position: [0, 1, 5], fov: 45 }}
          gl={{ antialias: true, powerPreference: 'default' }}
        >
          <PerformanceMonitor
            onDecline={() => setPerformanceTier('low')}
            onIncline={() => setPerformanceTier('high')}
            flipflops={3}
            onFallback={() => setPerformanceTier('low')}
          >
            <AdaptiveDpr pixelated />
          </PerformanceMonitor>
          <ambientLight intensity={0.5} />
          <directionalLight position={[5, 5, 5]} intensity={1} />
          <Suspense fallback={null}>
            <PresentationControls
              global
              snap
              rotation={[0, 0, 0]}
              polar={[-Math.PI / 4, Math.PI / 4]}
              azimuth={[-Math.PI / 4, Math.PI / 4]}
            >
              <RobotModel />
            </PresentationControls>
          </Suspense>
        </Canvas>
      </>
    );
  }
  ```
